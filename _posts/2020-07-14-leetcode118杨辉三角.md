### 问题描述

给定一个非负整数 numRows，生成杨辉三角的前 numRows 行。  

#### 示例 1:
```
输入: 5
输出:
[
     [1],
    [1,1],
   [1,2,1],
  [1,3,3,1],
 [1,4,6,4,1]
]
```

#### 分析：  
```
func generate(numRows int) [][]int {
    if 0 == numRows {
        return [][]int{}
    }
    res := make([][]int, numRows)
    for i := 0; i < numRows; i++ {
        res[i] = make([]int, i+1)
    }
    res[0][0] = 1
    for i := 1; i < numRows; i++ {
        res[i][0], res[i][i] = 1, 1
        for j := 1; j < len(res[i])-1; j++ {
            res[i][j] = res[i-1][j-1] + res[i-1][j]
        }
    }
    return res
}
```
